# QuizRoom

Quiz game which download questions from API

## Demo

https://oskartrapczynski.github.io/quizroom

## Installation

Install quizroom with npm

```
  npm install quizroom
  cd quizroom
```

For development

```
    npm run dev
```

For production

```
    npm run build
    deploy
```
